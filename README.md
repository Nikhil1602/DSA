### ‚≠ê Famous Algorithms & its uses ‚≠ê

1. [Sieve of Eratosthenes](https://www.youtube.com/watch?v=FERLTDTmxFM)
    - Useful to find all the prime numbers till n.
2. [Euclid‚Äôs Algorithm](https://www.youtube.com/watch?v=yHwneN6zJmU)
    - A classic algorithm to compute the greatest common divisor (GCD) of two numbers.
3. [Kadane's Algorithm](https://www.youtube.com/watch?v=lq8KOs1Ujas)
    - Useful in finding maximum sum among all the continous subarrays.
4. [Boyer-Moore's Algorithm](https://www.youtube.com/shorts/IyxdywkNdQk)
    - An efficient string searching algorithm to find the occurrence of pattern within larget text.
5. [Knuth-Morris-Pratt (KMP) Algorithm](https://www.youtube.com/watch?v=q4_90fOoS-s)
    - Useful in string-searching algorithm to find the all the occurrence of pattern within text.
6. [Rabin-Karp Algorithm](https://www.youtube.com/watch?v=yFHV7weZ_as)
    - Uses hashing to find any one of the set of pattern strings in a text.
7. [Floyd's Cycle Detection Algorithm](https://www.youtube.com/watch?v=5rNovVFB0TA)
    - Useful in cycle detection in linked list or sequence using two pointers moving at different speed (Tortoise and Hare).
8. [Dijkstra's Algorithm](https://www.youtube.com/watch?v=CmIQ29cUGiE)
    - Finds the shortest path from a source node to all other nodes in a graph with non-negative edge weights.
9. [Bellman-Ford Algorithm](https://www.youtube.com/watch?v=B5PmlJACZ9Y)
    - Computes shortest paths in a weighted graph, capable of handling negative edge weights.
10. [Floyd-Warshall Algorithm](https://www.youtube.com/watch?v=sLg6Leb-xt0)
    - Finds shortest paths in a weighted graph with positive or negative edge weights (all-pairs shortest path).
11. [Kruskal‚Äôs Algorithm](https://www.youtube.com/watch?v=8HeLu8wuLqo)
    - Finds the Minimum Spanning Tree (MST) for a connected weighted graph.
12. [Prim‚Äôs Algorithm](https://www.youtube.com/watch?v=EHRqQBlZAtU)
    - Another greedy algorithm to find the Minimum Spanning Tree (MST).
