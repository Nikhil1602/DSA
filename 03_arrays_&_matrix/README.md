There are two types of arrays in programming: `1D` arrays and `2D` arrays `(Matrix)`. For any N sized array, the indexing would always starts from `0` to `N-1`. There were operation on array that includes `traversal`, `insertion`, `deletion`, `searching`, `modification`, `sorting`, and `merging`.

## Built-in methods in arrays

- `[]`: to access the array element at any index given between `[` and `]`. e.g. `arr[0]`.
- `length`: return the size of the array.
- `push(value)`: push the element at last in the array.
- `pop()`: removes & returns the element from the last in array.
- `shift()`: removes & returns the element from the first in array.
- `unshift(value)`: push the element at first in the array.
- `includes(searched-element)`: to check whether the searched-element is present in the array or not.
- `reverse()`: to reverse the array elements.
- To know more about such methods visit [here](https://www.w3schools.com/js/js_array_methods.asp).
- To know about string methods visit [here](https://www.w3schools.com/js/js_string_methods.asp).

## Tricks & Tips

- To print the `n` elements cyclically without resetting the `i` of loop to 0, we should use `i % n`.
